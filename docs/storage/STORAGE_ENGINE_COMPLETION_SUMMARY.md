### 架构设计

1. **分层架构**
   ```
   Row Format Layer (行格式层)
   ↓
   Segment & Extent Layer (段和区层)
   ↓
   Tablespace Layer (表空间层)
   ↓
   Page Layer (页面层)
   ```

2. **策略模式**
   - 页面分配策略可配置
   - 表空间扩展策略可选
   - Extent复用策略灵活
   - 段分配策略按类型

3. **并发控制**
   - RWMutex读写分离
   - atomic操作原子统计
   - 后台协程异步处理

4. **缓存优化**
   - 统计缓存（5分钟过期）
   - 压缩结果缓存
   - 溢出数据缓存
   - 位图缓存（LRU）
