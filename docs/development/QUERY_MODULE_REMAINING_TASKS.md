## 五、建议实现顺序（查询模块）

1. **OPT-001 + OPT-002**：ICP 与覆盖索引 — 对简单查询与索引场景收益最大。
2. **OPT-016 + OPT-017**：统计信息与选择性 — CBO 和连接顺序/代价估算的基础。
3. **OPT-011 + OPT-012**：谓词下推与列裁剪完善 — 与执行器配合，减少中间数据量。
4. **OPT-003、OPT-004、OPT-013、OPT-014**：索引合并、前缀匹配、聚合消除、子查询 — 按业务需求排期。
5. **EXE-004、EXE-010**：SortMergeJoin、批量处理 — 大表 JOIN 与吞吐优化。
