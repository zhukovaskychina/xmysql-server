## 📝 技术笔记

### 关键技术点

1. **Redo 日志幂等性**
   - 使用 LSN 检查确保可重复执行
   - 页面 LSN >= 日志 LSN 时跳过

2. **版本链遍历**
   - 从最新版本开始遍历
   - 使用 ReadView.IsVisible() 判断可见性
   - 返回第一个可见版本

3. **MVCC 可见性规则**
   - 规则1: 当前事务创建的版本可见
   - 规则2: txID < lowWaterMark 可见
   - 规则3: txID >= highWaterMark 不可见
   - 规则4: 在活跃列表中不可见
