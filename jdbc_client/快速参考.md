## 💡 提示

### 测试前检查

1. ✅ XMySQL Server运行在 `localhost:3309`
2. ✅ 数据库用户名：`root`，密码：`root@1234`
3. ✅ Java 17或更高版本
4. ✅ Maven 3.6+

### 常见问题

**Q: 测试失败怎么办？**
A: 检查数据库连接配置和XMySQL Server是否正常运行

**Q: 如何只运行某一类测试？**
A: 使用 `-Dtest=类名` 参数，例如：`mvn test -Dtest=DDLOperationsTest`

**Q: 如何查看详细的测试输出？**
A: 查看 `target/surefire-reports/` 目录下的报告文件

**Q: 测试数据会影响生产数据吗？**
A: 不会，所有测试使用独立的测试数据库，测试后自动清理
