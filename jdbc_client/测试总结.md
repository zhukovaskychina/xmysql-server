## 🛠️ 使用的技术栈

### 测试框架

- **JUnit 5** (5.10.1) - 现代化的Java测试框架
  - `@Test` - 标记测试方法
  - `@BeforeAll/@AfterAll` - 类级别的初始化和清理
  - `@BeforeEach/@AfterEach` - 方法级别的初始化和清理
  - `@DisplayName` - 提供可读的测试名称
  - `@Order` - 控制测试执行顺序
  - `@ParameterizedTest` - 参数化测试

### 断言库

- **AssertJ** (3.24.2) - 流式断言库
  - 提供更好的可读性
  - 丰富的断言方法
  - 更清晰的错误消息

### 模拟框架

- **Mockito** (5.8.0) - 用于未来的单元测试扩展

### 构建工具

- **Maven Surefire Plugin** (3.2.3) - 运行测试并生成报告

## 🚀 快速开始

### 1. 安装依赖

```bash
cd jdbc_client
mvn clean install
```

### 2. 运行所有测试

```bash
# Linux/Mac
./run-tests.sh

# Windows
run-tests.bat

# 或使用Maven
mvn test
```

### 3. 运行特定测试

```bash
mvn test -Dtest=DDLOperationsTest
mvn test -Dtest=SelectQueryTest
```

### 4. 生成测试报告

```bash
mvn test
mvn surefire-report:report
```

报告位置：`target/surefire-reports/`
